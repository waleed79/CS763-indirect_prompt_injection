## Tier 3 Payload Objective

| Option | Description | Selected |
|--------|-------------|----------|
| Semantic naturalness | kimi generates factual-prose payloads with no injection markers; controlled anchor token ensures substring detection; qualitatively distinct from T1/T2 | ✓ |
| Persona/authority spoofing | Impersonate system voices ("SYSTEM NOTE:"); high raw ASR (89.6% in literature) but overlaps with Tier 1 [SYSTEM]: framing | |
| Soft persuasion / hypothetical framing | Embed instruction as plausible rhetoric; most realistic threat model; high risk of near-zero ASR on small LLMs | |

**User's choice:** Semantic naturalness (Recommended)
**Notes:** Anchor token (`t3.evilcorp.io`) ensures cross-tier ASR comparability while preserving the naturalness objective.

## Tier 4 Fragmentation Design

| Option | Description | Selected |
|--------|-------------|----------|
| 3 fragments | A=topic opener, B=benign connector, C=instruction completion. Strongest novelty claim; directly enables Phase 3.2 leave-one-out causal attribution ablation | ✓ |
| 2 fragments (continuation-signal) | A ends with semantic hook; B completes instruction. Higher co-retrieval probability but weaker research contribution | |
| 2 fragments (cross-reference) | "See also: Note [2]" pattern; reliable reassembly but footnote marker is a detectable heuristic | |

**User's choice:** 3 fragments (Recommended)
**Notes:** Co-retrieval rate is reported as an empirical finding, not assumed.
