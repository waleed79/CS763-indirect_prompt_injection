# Phase 02.2 Plan 02: Evaluate Attack Pipeline Summary

## What Was Built

Created `scripts/evaluate_attack.py`, an end-to-end evaluation harness that:
1. Loads config and creates a frozen-safe poisoned collection config via `dataclasses.replace`
2. Builds a `RAGPipeline` against `data/corpus_poisoned.jsonl` (520 passages: 500 clean + 20 poisoned)
3. Runs all 10 TEST_QUERIES through the pipeline
4. Records per-query: `tier1_retrieved`, `tier2_retrieved`, `retrieved_poisoned`, `hijacked_tier1`, `hijacked_tier2`, `hijacked`
5. Computes aggregate ASR metrics and writes `logs/attack_baseline.json`
