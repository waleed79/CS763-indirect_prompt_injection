## Decisions Made

- **MS-MARCO over NQ**: MS-MARCO's passage-retrieval task mirrors real RAG deployment more closely. NQ pairs are QA-format; MS-MARCO has web-search passages which are longer and more heterogeneous — better stress-test for retrieval.
- **Instruction smuggling over HTTP exfiltration**: The original Tier-2 design embedded `http://attacker.com/exfil?q=...` URLs, which produced 0% ASR because local Ollama models cannot make HTTP requests. Instruction smuggling — hiding the injection inside what looks like an editorial/audit note — tests whether the LLM follows injected in-context instructions.
- **5 Tier-1 variants**: A single "IGNORE ALL PREVIOUS INSTRUCTIONS" template was too narrow for a security paper. Five stylistically different variants (classic, system-tag, authority-framing, role-injection, markdown-header) provide per-style ASR breakdown and demonstrate attack diversity.
- **Passage IDs 20000+**: With 1000 clean passages (IDs 0–999), poisoned IDs starting at 10000 would still be safe, but 20000 provides a larger buffer and signals unambiguously that these are adversarial documents.
