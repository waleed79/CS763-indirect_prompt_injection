**FusedDefense class:**

```python
class FusedDefense:
    def __init__(self, models_dir: str = "models", threshold: float = 0.5):
        self.threshold = threshold
        self._models_dir = Path(models_dir)
        self._load_bert()      # loads self._bert_model, self._bert_tokenizer
        self._load_gpt2()      # loads self._gpt2_model, self._gpt2_tokenizer
        self._load_lr()        # loads self._lr, self._scaler
        self._load_signal4_baseline()  # loads self._s4_baseline = {"mu": float, "std": float}
```

`_load_bert`: load DistilBertTokenizerFast and DistilBertForSequenceClassification from `self._models_dir / "bert_classifier"`. Call `.eval()` on the model. Set `torch.set_grad_enabled(False)` or use `torch.no_grad()` in inference. Raise FileNotFoundError if bert_classifier/ directory is empty or missing the config.json.

`_load_gpt2`: load GPT2TokenizerFast and GPT2LMHeadModel from "gpt2" (uses HF cache). Call `.eval()`. Set pad_token_id = eos_token_id on tokenizer to suppress warnings.

`_load_lr`: call `_load_lr_from_json(str(self._models_dir / "lr_meta_classifier.json"))`.

`_load_signal4_baseline`: read `self._models_dir / "signal4_baseline.json"` as JSON dict with keys "mu" and "std". Store as `self._s4_baseline`.

`_bert_prob(text: str) -> float`: tokenize text with self._bert_tokenizer (truncation=True, max_length=256, return_tensors='pt'). Forward pass through self._bert_model inside torch.no_grad(). Apply softmax to logits[0]. Return float of index-1 probability (injection class).

`_perplexity_z(text: str) -> float`: call `_compute_perplexity_max_nll(text, self._gpt2_model, self._gpt2_tokenizer)`. This returns raw max NLL — the "z" in the name refers to how it is used as a feature (raw NLL, StandardScaler will normalize it before LR).

`_imperative_ratio(text: str) -> float`: call `_imperative_ratio(text)` module-level function.

`_retrieval_z(score: float) -> float`: compute z-score using self._s4_baseline: `(score - mu) / max(std, 1e-9)`.

`score(chunk: dict) -> dict`: extract text = chunk.get("text", "") and score = chunk.get("score", 0.0). Compute all 4 signals. Return {"bert_prob": float, "perplexity_z": float, "imperative_ratio": float, "retrieval_z": float}.

`_is_injection(chunk: dict) -> bool`: call self.score(chunk), build numpy feature array [[bert_prob, perplexity_z, imperative_ratio, retrieval_z]], apply self._scaler.transform(), call self._lr.predict_proba()[:,1][0], compare to self.threshold.

`filter(chunks: list[dict]) -> list[dict]`: return [c for c in chunks if not self._is_injection(c)].

**SingleSignalDefense class:**

```python
class SingleSignalDefense:
    VALID_SIGNALS = ("bert", "perplexity", "imperative", "fingerprint")

    def __init__(self, signal: str, models_dir: str = "models", threshold: float | None = None):
        if signal not in self.VALID_SIGNALS:
            raise ValueError(f"signal must be one of {self.VALID_SIGNALS}, got {signal!r}")
        self.signal = signal
        self._models_dir = Path(models_dir)
        self._threshold = threshold  # None = use signal-specific default
        self._init_signal()
```

`_init_signal`: conditionally load only what the signal needs:
- "bert": load DistilBERT (same as FusedDefense._load_bert). Default threshold = 0.5.
- "perplexity": load GPT-2 (same as FusedDefense._load_gpt2). Default threshold = 4.5 (NLL threshold, empirically: T1=4.64, T2=5.33, clean=4.49 — use 4.5 as conservative split).
- "imperative": no model load needed. Default threshold = 0.1 (any sentence with imperative verb = flagged).
- "fingerprint": load signal4_baseline.json. Default threshold = 2.0 (z-score of 2.0 catches targeted poison).

`_score_single(chunk: dict) -> float`: return just the relevant signal score using the same private methods as FusedDefense.

`filter(chunks: list[dict]) -> list[dict]`: return [c for c in chunks if self._score_single(c) <= (self._threshold or self._default_threshold)].

**Note on missing models:** When bert_classifier/ has no config.json (models not yet trained), _load_bert should raise FileNotFoundError with message: "BERT classifier not found at {path}. Run scripts/train_defense.py first." Same pattern for lr_meta_classifier.json and signal4_baseline.json.
