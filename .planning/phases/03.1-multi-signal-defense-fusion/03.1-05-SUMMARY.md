## Decisions Made

- `DEFENSE_CHOICES` constant at module level rather than inline in `argparse.add_argument` — consistent with existing `TIER*_ID_START` pattern in the file; enables future import by test code

- FPR measured only over `paired=False` queries — paired queries are attack queries where chunk removal is expected/desirable, not false positives
