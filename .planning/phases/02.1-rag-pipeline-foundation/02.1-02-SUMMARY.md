### `rag/corpus.py` (RAG-02)

- `Passage(NamedTuple)` — `passage_id`, `query`, `text`
- `load_nq_passages(n=500, seed=42)` — loads from `sentence-transformers/natural-questions` (44 MB); deduplicates on answer text; passes `seed` explicitly to `dataset.shuffle()` (not global numpy seed — Pitfall 3 in RESEARCH.md)
- `chunk_text(text, chunk_words=200, overlap_words=25)` — word-budget chunker; handles empty input, single-chunk short passages, multi-chunk overlapping windows; all words covered
- `persist_passages(passages, path="data/nq_500.jsonl")` — writes canonical JSONL
- `load_passages_from_jsonl(path)` — fast offline reload (no HuggingFace download)

## Design Decisions

1. **Word-budget chunking** (200 words, not exact tokens) — preserves full determinism, avoids MiniLM truncation (RESEARCH.md Pitfall 2). Documented in module docstring.
2. **Frozen Config** — runtime mutation caught immediately as `FrozenInstanceError`.
3. **Persisted JSONL corpus** — guarantees reproducibility even if HuggingFace dataset is updated (RESEARCH.md Open Question 3).
4. **Explicit seed to dataset.shuffle()** — HuggingFace ignores global numpy seed (RESEARCH.md Pitfall 3).
