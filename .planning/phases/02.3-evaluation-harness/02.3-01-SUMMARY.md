## Decisions Made

- URL payload chosen over EXFILTRATED token: local LLMs (mistral:7b, llama3.2:3b) do not echo parenthetical metadata tokens faithfully; compliance-framed URL instructions are echoed reliably, producing non-zero Tier-2 ASR
- attack_queries filter added to build_poisoned_passages(): only attack-paired queries (paired=True) get poisoned passages; clean queries (paired=False) produce no poisoned doc, enabling clean FPR measurement
- Tier-2 passage_id range shifted to 20050-20099 to accommodate 50 topics without overlapping Tier-1 range 20000-20049
