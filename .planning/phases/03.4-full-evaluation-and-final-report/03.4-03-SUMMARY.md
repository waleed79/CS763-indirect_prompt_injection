## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] matplotlib rejects `.tmp` extension in atomic save**

- **Found during:** First `python scripts/make_figures.py --output-dir figures` run.
- **Issue:** `fig.savefig(tmp_path, ...)` where `tmp_path = "figures/fig1_arms_race.png.tmp"` raised `ValueError: Format 'tmp' is not supported`. matplotlib infers format from path extension and the trailing `.tmp` is treated as the extension, not the desired `.png`.
- **Fix:** Pass `format="png"` explicitly to `fig.savefig` inside the `save_atomic` helper. The `os.replace` step still produces `.png` output as intended.
- **Files modified:** `scripts/make_figures.py` (one block: `save_atomic` function, added `format="png"` kwarg + 1-line comment).
- **Verification:** All 5 figures render successfully on retry; pytest 3/3 PASSED.
- **Committed in:** `ceec13f` (Task 1 commit) -- fix is part of the same task, pre-commit.
